📚 STRIP Function Example

SAS STRIP

Syntax:

STRIP(string)

Example:

data example;
  text = "  SAS Programming  ";
  stripped = strip(text);
run;

proc print data=example; run;

Explanation: Removes leading and trailing blanks from a string.

text = " SAS Programming "
stripped = "SAS Programming"

R stringr::str_trim

Syntax:

str_trim(string, side = "both")

Example:

library(stringr)
text <- "  SAS Programming  "
stripped <- str_trim(text)
stripped

Explanation: Removes leading and trailing spaces. `side = "both"` removes from both ends.

R Library: stringr must be installed using:
install.packages("stringr")
stripped = "SAS Programming"
