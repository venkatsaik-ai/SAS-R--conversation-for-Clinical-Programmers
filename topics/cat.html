📚 CAT Function Example

SAS CAT

Syntax:

CAT(argument1, argument2, ...)

Example:

data example;
  text1 = "SAS";
  text2 = "R";
  combined = cat(text1, text2);
run;

proc print data=example; run;

Explanation: Concatenates strings without adding any delimiter. Simply joins the values together.

text1 = SAS
text2 = R
combined = SASR

R paste0

Syntax:

paste0(...)

Example:

text1 <- "SAS"
text2 <- "R"
combined <- paste0(text1, text2)
data.frame(text1, text2, combined)

Explanation: Joins strings together **without any separator**, equivalent to SAS CAT.

text1 = SAS
text2 = R
combined = SASR
