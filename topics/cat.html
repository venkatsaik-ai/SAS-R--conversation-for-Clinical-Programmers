📚 CAT Function Example

SAS CAT

Syntax:

CAT(argument-1, argument-2, ...)

Example:

data example;
  first = "Hello";
  second = "World";
  combined = cat(first, second);
run;

proc print data=example; run;

Explanation: Concatenates multiple character strings without any delimiter.

first = Hello
second = World
combined = HelloWorld

R stringr::str_c

Syntax:

str_c(string1, string2, ..., sep = "")

Example:

library(stringr)
first <- "Hello"
second <- "World"
combined <- str_c(first, second)
combined

Explanation: Concatenates multiple strings. `sep=""` ensures no extra character between strings.

R Library: stringr must be installed using:
install.packages("stringr")
combined = HelloWorld
