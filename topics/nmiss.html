📚 NMISS Function Example

SAS NMISS

Syntax: NMISS(var1, var2, ...)

data example;
x = 10;
y = .;
z = 5;
missing_count = nmiss(of x y z);
run;

proc print data=example; run;

Explanation: Counts the number of missing numeric values among the specified variables.

Syntax: sum(is.na(c(var1, var2, ...)))

x <- 10
y <- NA
z <- 5
missing_count <- sum(is.na(c(x, y, z)))
missing_count
