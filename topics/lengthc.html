📚 LENGTHC Function Example

SAS LENGTHC

Syntax: LENGTHC(string)

data example;
text = "SAS Programming   ";
lenc = lengthc(text);
run;

proc print data=example; run;

Explanation: Returns the number of **characters including trailing blanks**.

lenc = 18
