📚 COMPRESS Function Example

SAS COMPRESS

Syntax:

COMPRESS(string, <characters-to-remove>)

Example:

data example;
  text = "S A S 123";
  compressed = compress(text);       /* removes spaces by default */
  compressed2 = compress(text, "S"); /* removes spaces and 'S' */
run;

proc print data=example; run;

Explanation: Removes specified characters from a string. By default, removes blanks.

text = S A S 123
compressed = SAS123
compressed2 = A123

R gsub

Syntax:

gsub(pattern, replacement, string)

Example:

text <- "S A S 123"
compressed <- gsub(" ", "", text)      # removes spaces
compressed2 <- gsub("[ S]", "", text)  # removes spaces and 'S'
data.frame(text, compressed, compressed2)

Explanation: Uses regular expressions to remove specific characters. Space (" ") removes blanks, "[ S]" removes spaces and 'S'.

text = S A S 123
compressed = SAS123
compressed2 = A123
