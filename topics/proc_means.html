📊 PROC MEANS Example

PROC MEANS

Syntax: PROC MEANS DATA=dataset; VAR var1 var2; RUN;

data example;
input age height weight;
datalines;
23 170 65
30 180 75
28 175 70
;
run;

proc means data=example mean std min max;
var age height weight;
run;

Explanation: Calculates descriptive statistics (mean, std, min, max) for numeric variables.

age mean=27
height mean=175
weight mean=70

R dplyr::summarise

Install required library:
install.packages("dplyr")

Syntax: summarise(data, mean(var1), sd(var1), min(var1), max(var1))

library(dplyr)

example <- data.frame(
  age = c(23,30,28),
  height = c(170,180,175),
  weight = c(65,75,70)
)

example %>%
  summarise(
    mean_age = mean(age),
    std_age = sd(age),
    min_age = min(age),
    max_age = max(age)
  )
mean_age = 27
std_age ≈ 3.6
min_age = 23
max_age = 30
