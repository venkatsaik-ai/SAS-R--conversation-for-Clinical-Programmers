📚 INTNX Function Example

SAS INTNX

Syntax: INTNX('interval', start-date, increment)

data example;
start = '01JAN2026'd;
next_month = intnx('month', start, 1);
next_week = intnx('week', start, 1);
format start next_month next_week date9.;
run;

proc print data=example; run;

Explanation: Returns the date shifted by a given interval (day, week, month, etc.).

next_month = 01FEB2026
next_week = 08JAN2026

R lubridate::%m+%

Install required library:
install.packages("lubridate")

Syntax: start %m+% months(n) / start + weeks(n)

library(lubridate)

start <- ymd("2026-01-01")
next_month <- start %m+% months(1)
next_week <- start + weeks(1)

next_month
next_week
next_month = "2026-02-01"
next_week = "2026-01-08"
