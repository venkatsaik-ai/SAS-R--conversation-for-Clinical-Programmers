📚 UPCASE Function Example

SAS UPCASE

Syntax: UPCASE(string)

data example;
text = "Sas Programming";
up_text = upcase(text);
run;

proc print data=example; run;

Explanation: Converts all characters to uppercase.

up_text = "SAS PROGRAMMING"

R toupper

Syntax: toupper(string)

text <- "Sas Programming"
up_text <- toupper(text)
up_text
up_text = "SAS PROGRAMMING"
