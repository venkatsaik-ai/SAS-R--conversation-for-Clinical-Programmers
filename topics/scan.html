📚 SCAN Function Example

SAS SCAN

Syntax:

SCAN(string, n, <delimiters>)

Example:

data example;
  text = "SAS R programming";
  word1 = scan(text, 1);
  word2 = scan(text, 2);
  word3 = scan(text, 3);
run;

proc print data=example; run;

Explanation: Extracts the nth word from a string. Default delimiter is space.

word1 = SAS
word2 = R
word3 = programming

R stringr::word

Syntax:

word(string, n)

Example:

library(stringr)
text <- "SAS R programming"
word1 <- word(text, 1)
word2 <- word(text, 2)
word3 <- word(text, 3)
data.frame(word1, word2, word3)

Explanation: Extracts the nth word from a string. Uses space as default delimiter.

word1 = SAS
word2 = R
word3 = programming
