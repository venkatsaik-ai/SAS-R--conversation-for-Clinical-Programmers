📚 LOWCASE Function Example

SAS LOWCASE

Syntax: LOWCASE(string)

data example;
text = "SAS Programming";
low_text = lowcase(text);
run;

proc print data=example; run;

Explanation: Converts all characters to lowercase.

low_text = "sas programming"

R tolower

Syntax: tolower(string)

text <- "SAS Programming"
low_text <- tolower(text)
low_text
low_text = "sas programming"
