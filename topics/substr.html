📚 SUBSTR Function

📋 Function Syntax

SAS SUBSTR

SUBSTR(source, start [, length])
SUBSTR(source, start, length) = value  /* For assignment */

Parameters:

  • source - Character variable/string to extract from
  • start - Starting position (1-indexed)
  • length - Number of characters to extract (optional)

Returns: Character substring

R substr

substr(x, start, stop)
substr(x, start, stop) <- value  /* For assignment */

Parameters:

  • x - Character string/vector to extract from
  • start - Starting position (1-indexed)
  • stop - Ending position (inclusive, 1-indexed)

Returns: Character substring/vector

📖 Detailed Explanation

The SUBSTR/substr function extracts a portion of a character string based on starting position. This is one of the most commonly used functions in clinical programming for parsing identifiers, dates, and coded values.

⚠️ Critical Difference:
  • SAS: Uses length parameter (number of characters to extract FROM the start position)
  • R: Uses stop parameter (ending position, not length)
  • Conversion Formula: R_stop = SAS_start + SAS_length - 1

✅ Similarities:

  • Both use 1-based indexing (first character is at position 1, not 0)
  • Both extract substrings from a specified position
  • Both return character results
  • Both support assignment/replacement operations
  • No external libraries required in either language

❌ Differences:

  • SAS uses length-based extraction; R uses position-based extraction
  • R is vectorized (works on all elements of a vector at once)
  • SAS operates primarily on single values in DATA steps

💡 Code Examples & Outputs

SAS Example 1: Basic Extraction

data example;
  string = "HelloWorld";
  substring = substr(string, 1, 5);
  put substring;
run;
Output: Hello

SAS Example 2: Extract Without Length

data example;
  string = "HelloWorld";
  substring = substr(string, 7);
  put substring;
run;
Output: World

SAS Example 3: Extract Middle Portion

data example;
  protocol = "STUDY2024-001";
  year = substr(protocol, 6, 4);
  site = substr(protocol, 11, 3);
  put year= site=;
run;
Output: year=2024 site=001

SAS Example 4: Assignment/Replacement

data example;
  string = "HelloWorld";
  substr(string, 1, 5) = "Hi   ";
  put string;
run;
Output: Hi World

R Example 1: Basic Extraction

string <- "HelloWorld"
substring <- substr(string, 1, 5)
print(substring)
Output: "Hello"

R Example 2: Extract From Position 7 to End

string <- "HelloWorld"
substring <- substr(string, 7, nchar(string))
print(substring)
Output: "World"

R Example 3: Extract Middle Portions (Vectorized)

protocol <- c("STUDY2024-001", "STUDY2024-002")
year <- substr(protocol, 6, 9)
site <- substr(protocol, 11, 13)
print(data.frame(year, site))
Output: year site 2024 001 2024 002

R Example 4: Assignment/Replacement (Vectorized)

string <- c("HelloWorld", "GoodbyeWorld")
substr(string, 1, 5) <- "Hi   "
print(string)
Output: "Hi World" "Hi byeWorld"

⚙️ Working Behavior Comparison

Aspect SAS R
Indexing 1-based (first character = position 1) 1-based (first character = position 1)
Position Parameters start, length (number of chars) start, stop (ending position)
Missing End Position Extracts to end of string automatically Must specify or use nchar()
Vector Operations Works on single values in DATA steps Fully vectorized - processes all elements
Assignment (Replacement) Direct: substr(x,1,5)="new" Using <-: substr(x,1,5) <- "new"
Out of Bounds Returns partial string or empty Returns partial string or empty string
Data Types Supported Character variables only Character vectors/strings
Library Required None (base SAS) None (base R)

🔄 SAS to R Conversion Guide

Key Formula for Conversion:

SAS:  substr(string, start, length)
R:    substr(string, start, start + length - 1)

Conversion Examples:

SAS Code R Equivalent Explanation
substr(x, 1, 3) substr(x, 1, 3) start=1, length=3 → start=1, stop=1+3-1=3
substr(x, 5, 4) substr(x, 5, 8) start=5, length=4 → start=5, stop=5+4-1=8
substr(x, 10) substr(x, 10, nchar(x)) Extract from position 10 to end
substr(date,1,4) substr(date, 1, 4) Extract year from YYYYMMDD format

🎯 Clinical Programming Use Cases

  • Extract Patient ID Prefix: site_code <- substr(subject_id, 1, 2) (extracts first 2 characters)
  • Extract Date Components: year <- substr(visit_date, 1, 4) (from YYYY-MM-DD format)
  • Parse Protocol Number: protocol_year <- substr(protocol_id, 1, 4) (e.g., "2024" from "2024-ABC-001")
  • Extract Visit Code: visit <- substr(record_id, 6, 2) (specific positions in ID)
  • Extract AE Severity Code: severity <- substr(ae_code, 5, 1) (last digit indicates severity)
  • Parse Lab Test Codes: lab_domain <- substr(test_code, 1, 3) (chemistry, hematology, etc.)

⚡ Performance Notes

  • SAS: Very efficient for character manipulation; use in DATA steps or in PROC SQL
  • R: Also efficient; vectorization makes it faster for multiple strings
  • Alternative in R: stringr::str_sub() from the stringr package offers similar functionality
  • Consider: For clinical data with thousands of records, R vectorization often outperforms SAS
