📚 SUBSTR / substr Function Example

SAS SUBSTR

Syntax:

SUBSTR(source, start [, length])
SUBSTR(source, start, length) = value  /* For assignment */

Example:

data example;
  string = "HelloWorld";
  substring = substr(string, 1, 5);
  put substring;
run;

Explanation: Extracts the first 5 characters from string.

Output: Hello

R substr

Syntax:

substr(x, start, stop)
substr(x, start, stop) <- value  /* For assignment */

Example:

string <- "HelloWorld"
substring <- substr(string, 1, 5)
print(substring)

Explanation: Extracts the first 5 characters from string in R.

Output: "Hello"
