📚 RIGHT Function Example

SAS RIGHT

Syntax:

RIGHT(string)

Example:

data example;
  text = "SAS Programming   ";
  right_text = right(text);
run;

proc print data=example; run;

Explanation: Shifts all trailing blanks in a string so the last non-blank character is at the end.

text = "SAS Programming "
right_text = "SAS Programming"

R substr / stringr

Syntax:

substr(string, start, stop)

Example:

text <- "SAS Programming   "
right_text <- sub("\\s+$", "", text)  # removes trailing spaces
right_text

Explanation: Removes trailing spaces using regular expressions to replicate SAS RIGHT behavior.

No extra library required for base R.
Optional: stringr::str_trim(text, side="right")
right_text = "SAS Programming"
