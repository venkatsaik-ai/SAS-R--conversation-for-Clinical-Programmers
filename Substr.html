SUBSTR Function

SAS Syntax & Example

/* Example SAS Code */
data example;
  string = "HelloWorld";
  sub_str = substr(string,1,5);
run;

R Syntax & Example

# Example R Code
string <- "HelloWorld"
sub_str <- substr(string,1,5)
sub_str
# Output: "Hello"

Behavior Comparison

In SAS, substr extracts a substring starting at a specific position. In R, substr works the same. No extra library is required in R.
