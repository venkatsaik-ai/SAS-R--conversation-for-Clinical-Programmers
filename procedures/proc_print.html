PROC PRINT – SAS vs R

Basic PROC PRINT

SAS PROC PRINT

data students;
input id name $ marks;
datalines;
1 John 85
2 Mary 90
3 David 88
;
run;

proc print data=students;
run;
Output Dataset
ID NAME MARKS
1 John 85
2 Mary 90
3 David 88

R Print Data Frame

students <- data.frame(
id=c(1,2,3),
name=c("John","Mary","David"),
marks=c(85,90,88)
)

print(students)
Output
ID Name Marks
1 John 85
2 Mary 90
3 David 88
PROC PRINT with VAR Statement

SAS PROC PRINT with VAR

proc print data=students;
var name marks;
run;
Output Dataset
NAME MARKS
John 85
Mary 90
David 88

R Selecting Columns

students[,c("name","marks")]
Output
Name Marks
John 85
Mary 90
David 88
