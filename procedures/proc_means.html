PROC MEANS – Basic Example

SAS : PROC MEANS

Syntax
PROC MEANS DATA=dataset;
VAR variable;
RUN; 

data example;
input age height weight;
datalines;
23 170 65
30 180 75
28 175 70
;
run;

proc means data=example mean std min max;
var age height weight;
run;

Explanation
PROC MEANS calculates descriptive statistics like mean, std, minimum and maximum.

age mean = 27
height mean = 175
weight mean = 70

R : dplyr summarise()

Install package 
install.packages("dplyr")
library(dplyr)

example <- data.frame(
age = c(23,30,28),
height = c(170,180,175),
weight = c(65,75,70)
)

example %>%
summarise(
mean_age = mean(age),
std_age = sd(age),
min_age = min(age),
max_age = max(age)
)
mean_age = 27
std_age ≈ 3.6
min_age = 23
max_age = 30
PROC MEANS with CLASS Statement

SAS : PROC MEANS with CLASS

Syntax
PROC MEANS DATA=dataset;
CLASS variable;
VAR variable;
RUN; 

data example;
input gender $ age height weight;
datalines;
M 23 170 65
F 30 180 75
M 28 175 70
F 25 165 60
;
run;

proc means data=example mean min max;
class gender;
var age weight;
run;

Explanation
CLASS groups the data and calculates statistics for each group.

Gender M → age mean = 25.5
Gender F → age mean = 27.5

R : group_by() + summarise()

Install package 
install.packages("dplyr")
library(dplyr)

example <- data.frame(
gender = c("M","F","M","F"),
age = c(23,30,28,25),
height = c(170,180,175,165),
weight = c(65,75,70,60)
)

example %>%
group_by(gender) %>%
summarise(
mean_age = mean(age),
min_age = min(age),
max_age = max(age)
)
M → mean_age = 25.5
F → mean_age = 27.5
